Points To Remember

  • You can encrypt plain text password to MD5 hash using 
    • MessageDigest 
    • Apache Common Codec - DigestUtils
  • You can also add your own salt to make your password harder to decode.
  • Package java.security contains classes that can be used to encrypt passwords.
  • The data to be encoded is often called message and the encoded hash is called message Digest or simply digest.
