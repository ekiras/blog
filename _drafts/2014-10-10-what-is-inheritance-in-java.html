Points To Remember

  • Inheritance is always achieved with two classes, a base class and a parent class.
  • Base class gets all non private methods and variables of parent class.
  • We can override all the non private methods of a parent class in a base class.
  • Variables does not get overridden.
  • We cannot override a static method.
  • We cannot override a private method of the parent class in base class.
