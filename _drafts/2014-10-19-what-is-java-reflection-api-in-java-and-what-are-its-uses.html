What is Java Reflection API
Java Reflection  API is a technique that enables us to inspect classes, interfaces, fields and methods at runtime, without knowing the names of the classes, methods etc. at compile time. It is also possible to instantiate new objects, invoke methods and get/set field values using reflection.

Reflection API was introduced in java jdk v1.1
