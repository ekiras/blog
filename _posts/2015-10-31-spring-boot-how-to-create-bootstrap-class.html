Points To Remember

In order to create a class that acts like a bootstrap for the application, that class needs to implement the InitializingBean of the package org.springframework.beans.factory.InitializingBean

How to create a Bootstrap class in spring boot application

Following is an example of a Bootstrap class. This class will be executed first when the application is coming up and is ready to server requests. Any class that implements InitializingBean will be executed before the application is up and its afterPropertiesSet method will be called.
