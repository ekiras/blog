 class A {
        int i;
        void display() {
            System.out.println(i);
        }
    }    
    class B extends A {
        int j;
        void display() {
            System.out.println(j);
        }
    }    
    class inheritance_demo {
        public static void main(String args[])
        {
            B obj = new B();
            obj.i=1;
            obj.j=2;   
            obj.display();     
        }
   }
0 1
2 3
C
This is a case of method overriding or dynamic polymorphism. Here the method display of class A gets overridden by the display method of class B. Thus when the call for display method is made, method of class B gets executed.
