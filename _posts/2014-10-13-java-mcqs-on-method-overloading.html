  return a * b;
 }

  Test obj = new Test();

  obj.area(1F, 2.2);
  obj.area(1F, 2);
 }
}
double float float int double float

D
1 is an in value, 2.2 is a double value, 1F is a float value and 2 is int. So area(1,2.2) will go to area(double,double), area(1F,2.2) will also go to area(double,double) and area(1F,2) will go to area(float,float)
