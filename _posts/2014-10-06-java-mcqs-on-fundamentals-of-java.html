Which of these is platform independent ?
bytecode JVM
both A and B none of these
A
bytecode is platform independent. JVM is platform dependent and there are different JVM's for different OS like windows, linux etc, but they all produce same bytecode, thus a bytecode once produced can be run on any operating system.
